Projeto de desenvolvimento de um aplicativo para Android que permita manipulacao e
preenchimento de formulários, que tem como objetivo de uso criacao de
pesquisas de satisfacao e também de linhas de credito.

Principais funcões do aplicativo são:
-Criacao de formularios.
-Edicão de formulários.
-Preenchimento e envio de aquisicão para possível base de dados.
